1. git 사이트에서 설치
2. 등록
git config --global user.name a94670681
git config --global user.name a94670681@gmail.com
3. git 으로 관리할 프로젝트 폴더로 이동해서 터미널 열기(or git bash)
4. git init 명령어로 git 저장소 생성
5. 파일 생성 후 저장(아무파일이나)
6. git add * 로 감시할 파일명으로 감시할 파일 추가
7. git commit -m "버전설명" 으로 저장소에 저장하기(파일 또는 스냅샷) -> git status 확인
8. git status로 저장할 수 있는 상태인지 확인하기
9. git log --oneline --decorate --graph --all 으로 커밋 이력 확인하기
10. git commit -a -m "설명" a는 git add * 과 같다.
11. git diff 로 파일 상태 비교(스냅샷이 찍힐 내역확인)
